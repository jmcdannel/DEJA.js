# Dashboard

The Monitor Dashboard (`/`) presents a real-time split-screen view of all system activity. Panels include DCC command logs, turnout state changes, effect triggers, sensor events, and device serial output.

Each panel can be expanded to full screen via the dedicated [Log Viewer](/docs/monitor/logs).
