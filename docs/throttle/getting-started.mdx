# Getting Started with Throttle

Prefer to watch? Here's the full walkthrough.

{/* VIDEO: Throttle app complete walkthrough — connect, add a loco, control speed, throw turnouts, fire a route, trigger effects */}

Throttle is the main app for controlling your model railroad. This guide walks you through a complete session — from connecting to your command station to running trains, throwing turnouts, and triggering effects.

Before you begin, make sure you've [installed DEJA.js](/docs/installation) and [set up your layout](/docs/quick-start).

---

## Connect

Open Throttle in your browser. The home screen shows the connection status of your command station. When the server is running and connected, the status indicator turns green.

![Throttle home screen showing connection status](/screenshots/throttle_mobile_home.png)

See [Connection Setup](/docs/throttle/connect) for details on connecting to your command station.

---

## Add a Locomotive

Tap the **+** button on the throttle list to add a locomotive to your session. Enter the DCC address and a name for the locomotive. You can also assign an image and roadname.

![Add locomotive dialog](/screenshots/throttle_desktop_add-loco.png)

See [Roster](/docs/throttle/roster) for more on managing your locomotive roster.

---

## Select a Locomotive

The throttle list shows all locomotives in your session. Tap a locomotive to open its throttle control screen.

![Throttle list with locomotives](/screenshots/throttle_desktop_throttle-list.png)

See [Throttle List](/docs/throttle/throttle-list) for grid view options and multi-train management.

---

## Control Speed & Direction

Use the speed controls to set your locomotive's speed. Tap the direction button to switch between forward and reverse. The speed display shows the current DCC speed value (0–126).

![Throttle control screen with speed slider](/screenshots/throttle_mobile_throttle.png)

See [Throttle Control](/docs/throttle/throttle) for slider controls, button controls, and swipe navigation between locomotives.

---

## Use Functions

Tap the function button to open the speed dial. Toggle decoder functions like headlights (F0), bell (F1), and horn (F2). Each function button shows its current state — on or off.

![Function speed dial expanded](/screenshots/throttle_mobile_functions.png)

See [Throttle Control](/docs/throttle/throttle) for the full list of function buttons (F0–F28).

---

## Throw Turnouts

The turnouts view shows all configured turnouts on your layout. Tap a turnout to throw or close it. The color changes to indicate the current state.

![Turnouts view with state indicators](/screenshots/throttle_desktop_turnouts.png)

See [Turnouts](/docs/throttle/turnouts) for card view, list view, and turnout configuration.

---

## Fire a Route

Routes execute a sequence of turnout changes with a single tap. Select a route from the list to run it. A progress bar shows the execution status as each turnout in the sequence is thrown.

![Routes view with route list](/screenshots/throttle_desktop_routes.png)

See [Routes](/docs/throttle/routes) for route configuration and track map integration.

---

## Try Effects

Trigger sound and lighting effects on your layout. The effects view shows all available effects grouped by category. Tap an effect to activate it.

![Effects view with available effects](/screenshots/throttle_mobile_effects.png)

See [Effects](/docs/throttle/effects) for effect types, categories, and configuration.
