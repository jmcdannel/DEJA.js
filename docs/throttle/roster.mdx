# Roster

The Roster screen (`/locos`) displays all locomotives configured for your layout. Each entry shows the locomotive's name, DCC address, road name, and color. Tap a locomotive to open its throttle control screen.

Locomotives are configured in the [Cloud app](/docs/cloud) and stored in Firebase Firestore at `layouts/{layoutId}/throttles`. The roster updates in real time as changes are made in the Cloud app.
