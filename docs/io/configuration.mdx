## рџЌ“ Pico W вЂ” `config.json`

Pin mapping from logical pin numbers to physical CircuitPython GPIO names.

```json
{
  "pins": {
    "6": "GP6",
    "7": "GP7",
    "8": "GP8",
    "9": "GP9"
  }
}
```

Keys are the logical pin numbers used in commands (`"pin": 8`). Values are CircuitPython GPIO names (`"GP8"`).
