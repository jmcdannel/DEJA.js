# Cloud App

The Cloud app is the administration hub for managing your DCC-EX layout. Configure locomotives, turnouts, routes, effects, signals, sounds, and connected devices from a centralized dashboard.

## Navigation

The left sidebar organizes features into three sections:

| Section | Items |
|---|---|
| **Modules** | Roster, Turnouts, Routes, Effects, Signals, Sounds |
| **Hardware** | Sensors, Devices, DCC-EX |
| **System** | Settings, Emulator |

A pinned footer row links to the other DEJA suite apps (Cloud, Throttle, Monitor, Tour).

## Key Features

- **Dashboard** — Overview of layout status, connected devices, and system health
- **Roster Management** — Add, edit, and organize locomotives with custom names, addresses, and function mappings; bidirectional sync with the DCC-EX CommandStation roster
- **Turnout Configuration** — Define turnouts with DCC addresses and friendly labels
- **Route Builder** — Create routes that set multiple turnouts in sequence
- **Sound Library** — Upload, browse, and manage sound files stored in Vercel Blob
- **Effects Management** — Configure sound effects, lights, and animations
- **Sound Library** — Upload and manage audio files for playback during effects
- **Signal Wiring** — Set up signal aspects and output pin mappings
- **Sensor Management** — Configure sensors and automation rules
- **Device Management** — Monitor and configure connected hardware devices
- **DCC-EX Console** — Direct command-line access to the DCC-EX command station
- **Settings** — Layout settings, user profile, and billing
- **Upgrade** — View subscription plans and upgrade options
- **404 Page** — Brand-consistent not-found page with links to Dashboard, Roster, and Effects

## Onboarding

New users are guided through an onboarding wizard to create their first layout. After completing onboarding, a **Setup Complete** page provides:

- Confirmation of layout creation with the user's Layout ID
- Quick-add form to register a first locomotive (DCC address + optional name)
- Server installation instructions with credentials
- Curated links to documentation (install guide, roster, turnouts, effects)
- Direct navigation to the dashboard

Layouts are automatically approved on creation — there is no admin approval step.

### Roster

![Roster Management](/screenshots/cloud_desktop_roster.png)

The Roster page manages your locomotive collection and supports **bidirectional DCC-EX CommandStation sync**:

- **Sync to DCC-EX** — Pushes all locos from the DEJA roster into the CommandStation's internal memory and saves to EEPROM. This makes your roster available in EX-WebThrottle and other native DCC-EX clients.
- **Import from DCC-EX** — Reads the CommandStation's roster and imports any locomotives that don't already exist in DEJA Cloud (merge, no overwrites).
- **Auto-sync on save** — When you edit a locomotive and click Save in the Edit Loco form, the entry is automatically pushed to the CommandStation.
- **Status feedback** — A snackbar displays sync progress (`Syncing…`, `Success`, or `Error`) via a live Firebase RTDB subscription.

## Technical Details

- **Framework:** Vue 3 with Composition API
- **State:** Firebase Firestore via VueFire
- **Auth:** Firebase Auth with onboarding wizard and auto-approved layouts
- **Routing:** Vue Router 4 with meta-based guard chain
- **Port:** 3011 (development)
