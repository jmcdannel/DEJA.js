# Throttle App

The Throttle app is the primary user-facing interface for controlling model trains. It provides real-time speed, direction, and function controls via a mobile-friendly web interface.

## Home

The home screen shows available locomotives and quick access to all control features.

![Throttle Home](/screenshots/throttle_desktop_home.png)

## Throttle Control

Control individual locomotives with a speed slider, direction toggle, and function buttons (lights, horn, bell, etc.).

![Throttle Control](/screenshots/throttle_desktop_throttle.png)

## Throttle List

View and manage all active throttle sessions. Select a locomotive to control or add a new one from the roster.

![Throttle List](/screenshots/throttle_desktop_throttle-list.png)

## Turnouts

Throw and close turnouts (track switches) with a visual interface showing current state.

![Turnouts](/screenshots/throttle_desktop_turnouts.png)

## Routes

Execute predefined routes that set multiple turnouts and signals in sequence.

![Routes](/screenshots/throttle_desktop_routes.png)

## Effects

Trigger sound effects, lighting sequences, and other layout animations.

![Effects](/screenshots/throttle_desktop_effects.png)

## Conductor View

A birds-eye layout view for dispatchers showing train positions and track status.

![Conductor View](/screenshots/throttle_desktop_conductor.png)

## Mobile Views

The throttle app is optimized for mobile devices with touch-friendly controls.

![Mobile Home](/screenshots/throttle_mobile_home.png)

![Mobile Throttle](/screenshots/throttle_mobile_throttle.png)

## Settings

Unified settings page with sections for Account, Billing & Subscription (Stripe plan management and billing portal), Appearance (dark/light/high-contrast themes and per-page background customization), Throttle configuration (speed steps), Connection settings (DEJA Server or WiThrottle protocol with host/port config), and Footer Menu Favorites.

## Technical Details

- **Framework:** Vue 3 with Composition API
- **UI:** Vuetify 3 + Tailwind CSS
- **State:** Pinia stores, Firebase Firestore via VueFire, localStorage via VueUse
- **Auth:** Firebase Auth with social providers (Google, GitHub, Facebook, Apple, Microsoft) and email/password
- **Billing:** Stripe subscription management with plan display and billing portal access
- **Real-time:** Device status synced across all apps via Firebase
- **Routing:** Vue Router 4 with auth guards
