# Connect

The Connect screen (`/connect`) lets you establish a connection to your DEJA.js server or a WiThrottle-compatible command station. Enter the server host and port, then connect to begin controlling your layout.

Once connected, the app can send DCC commands through the server to your CommandStation hardware.
