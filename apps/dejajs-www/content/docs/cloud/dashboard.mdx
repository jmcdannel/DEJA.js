# Dashboard

The Dashboard (`/`) is the Cloud app's home screen. It provides a system overview showing connected device status, layout summary statistics, and quick-access tiles for each configuration section (Roster, Turnouts, Routes, Effects, Signals, Sensors, and DCC-EX).

The dashboard requires authentication, a completed onboarding flow, admin approval, and a selected layout.
