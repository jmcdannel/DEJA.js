# Shared 404 Page Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a shared `NotFoundPage.vue` component in `@repo/ui` and wire it into all four Vue apps (cloud, throttle, monitor, tour).

**Architecture:** A single source-of-truth component lives in `packages/ui/`. Each app keeps a thin wrapper view (`NotFound.vue`) that passes app-specific nav links as props. Routers reference the local wrapper via lazy import; the wrapper imports the shared component from `@repo/ui`.

**Tech Stack:** Vue 3 `<script setup lang="ts">`, Vuetify 3 (`v-btn`), `@repo/ui` (Logo, shared component), Vue Router 4 catch-all routes, pnpm workspaces.

**Spec:** `docs/superpowers/specs/2026-03-16-shared-404-page-design.md`

## Chunk 1: Shared Component + Package Exports

### Task 1: Create `NotFoundPage.vue` in `@repo/ui`

**Files:**
- Create: `packages/ui/src/NotFoundPage.vue`

- [ ] **Step 1: Create the component**

```vue
<script setup lang="ts">
import Logo from './Logo.vue'

interface NavLink {
  label: string
  to: string // Internal vue-router paths only. External URLs are not supported.
  icon?: string
}

interface Props {
  links?: NavLink[]
}

const props = withDefaults(defineProps<Props>(), {
  links: () => [{ label: 'Go Home', to: '/' }],
})
</script>

<template>
  <div class="not-found-page">
    <Logo size="md" class="mb-8" />
    <p class="text-h1 font-weight-bold not-found-code mb-4">404</p>
    <p class="text-h5 mb-2 not-found-heading">Page not found.</p>
    <p class="text-body-2 not-found-sub mb-8">
      Looks like this train has left the station.
    </p>
    <div class="d-flex flex-wrap justify-center gap-3">
      <v-btn
        v-for="link in props.links"
        :key="link.to"
        :to="link.to"
        :prepend-icon="link.icon"
        variant="tonal"
        color="primary"
        size="large"
      >
        {{ link.label }}
      </v-btn>
    </div>
  </div>
</template>

<style scoped>
.not-found-page {
  min-height: 100vh;
  background-color: #0f172a;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  text-align: center;
  padding: 24px;
}

.not-found-code {
  color: #00e3ef;
  font-size: 6rem !important;
  line-height: 1;
}

.not-found-heading {
  color: #94a3b8;
}

.not-found-sub {
  color: #94a3b8;
}
</style>
```

- [ ] **Step 2: Add barrel export to `packages/ui/src/index.ts`**

Add after the `SplashPage` export line (it is currently the second line in the file):
```ts
export { default as NotFoundPage } from './NotFoundPage.vue'
```

- [ ] **Step 3: Add named export to `packages/ui/package.json`**

In the `"exports"` object, add after the `"./SplashPage"` entry:
```json
"./NotFoundPage": "./src/NotFoundPage.vue",
```

- [ ] **Step 4: Commit**

```bash
git add packages/ui/src/NotFoundPage.vue packages/ui/src/index.ts packages/ui/package.json
git commit -m "feat(ui): add shared NotFoundPage component"
```
